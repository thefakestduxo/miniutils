# Miniround Utils
This datapack adds a Grace Period timer as well as automatic Boogeyman Rolling for Minirounds!

To use this datapack, you first have to run `/function duxo:mini/config` which will pull up a GUI of all the Config Options (more will be added in future versions).

If you would like to manually change this (like go over the limit of 20) 
you can do `/function duxo:mini/setconfig {gtime:<Time you want for grace in minutes>,btime:<Time you want for Boogeyman in minutes>}`

In this GUI (so far) are:
  Grace Period Time (minutes) 
  Boogeyman Time (minutes)



 When you want to start your miniround you can either do `/session start` (like a boring person) or do `/function duxo:start` (like a cool individual)

It is recommended to put the resource pack (https://download.mc-packs.net/pack/c7a4ac785a06355c5c9a7db5010680cb1b36b52f.zip) on the server. If you don't know how to do that, Kasai Sora did a great tutorial on how!
